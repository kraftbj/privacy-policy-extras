== Description ==

Adds basic UI to adjust the privacy policy feature of WordPress 4.9.6+.

Options include setting a custom URL for the privacy policy and updating the "Privacy Policy" text.

Development is taking place at https://github.com/kraftbj/privacy-policy-extras/

== Installation ==

Via your site's wp-admin Dashboard:

1. Visit Plugins->Add New
1. Search for "Privacy Policy Extras"
1. Click Install to add the plugin to your server
1. Click Activate Plugin to turn this plugin on.
1. Visit Tools->Privacy to see the plugin's options.

== Frequently Asked Questions ==

= What is this? =

The privacy policy feature added in WordPress 4.9.6 has built-in options to use a custom URL or change the "Privacy Policy" text used in links, but no UI. This plugin adds UI.

= Why would I want a custom URL? =

If you're running a site that is a part of a larger organization, you may want to link to the parent organization's privacy policy.
